# AgroLumen — MYOSA Contest 2026 Blog Submission

This repository is the official MYOSA Contest 2026 blog submission for team **AgroLumen** (Universidad del Magdalena, Colombia).

**Project:** Smart Solar-Powered Light Trap for Monitoring and Control of *Spodoptera frugiperda* Using MYOSA Platform.

**Conference target:** IEEE BIOSENSORS (Bangkok) — APSCON 2026.

The blog post lives in [`agrolumen-spodoptera.md`](./agrolumen-spodoptera.md). All images and the demo video are local files in the same folder, per the MYOSA submission guidelines.

For the full firmware (ESP32) and web dashboard (Next.js + Neon) source code, see the main project repository.
